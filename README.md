# PKI Generator

PKI Generator is a Java 21 command-line tool for generating local PKI material, including root CAs, intermediate CAs, website certificates, ICP-Brasil-style CPF/CNPJ certificates, and cloned certificates signed by a local CA.

This project documents, in executable Java code, how to generate certificates using the Java libraries used here, especially Bouncy Castle and the JCA APIs.

## Safety Notice

The certificates, keys, default values, sample identity data, and local CA material generated by this project are intended for local development, testing, learning, and experimentation only. Do not use them for production systems, public trust, real ICP-Brasil operations, or sensitive environments.

## Requirements

- Java 21
- Maven 3.9+

## Build

```bash
mvn clean package
```

The executable shaded JAR is created at:

```bash
target/pkigen.jar
```

## Help

```bash
java -jar target/pkigen.jar --help
java -jar target/pkigen.jar ca --help
java -jar target/pkigen.jar website --help
java -jar target/pkigen.jar icpcpf --help
java -jar target/pkigen.jar icpcnpj --help
java -jar target/pkigen.jar clone --help
```

By default, generated files are written under:

```text
~/.pki-generator
```

You can override this with `--output-dir`.

## Examples

Generate a root CA:

```bash
java -jar target/pkigen.jar --output-dir ./pki ca --common-name "Local Root CA"
```

Generate an intermediate CA signed by the root CA:

```bash
java -jar target/pkigen.jar --output-dir ./pki ca --common-name "Local Intermediate CA" --issuer local_root_ca
```

Generate a website certificate signed by the root CA:

```bash
java -jar target/pkigen.jar --output-dir ./pki website --issuer local_root_ca --is-root true --common-name localhost --dns-names localhost,example.local --ip-addrs 127.0.0.1
```

Generate a website certificate signed by the intermediate CA:

```bash
java -jar target/pkigen.jar --output-dir ./pki website --issuer local_intermediate_ca --is-root false --common-name api.example.local --dns-names api.example.local,localhost --ip-addrs 127.0.0.1
```

Generate an ICP-Brasil-style CPF certificate:

```bash
java -jar target/pkigen.jar --output-dir ./pki icpcpf --issuer local_root_ca --is-root true --common-name "JOHN DOE" --cpf 11144477735 --email john.doe@example.com --cert-type A1
```

Generate an ICP-Brasil-style CNPJ certificate:

```bash
java -jar target/pkigen.jar --output-dir ./pki icpcnpj --issuer local_root_ca --is-root true --common-name "EXAMPLE COMPANY" --cnpj 11222333000181 --email contact@example.com --resp-name "JOHN DOE" --resp-cpf 11144477735 --cert-type A1
```

Clone an existing certificate and sign it with a local CA:

```bash
java -jar target/pkigen.jar --output-dir ./pki clone --source-cert ./server.crt --issuer local_root_ca --is-root true --name cloned_server
```
